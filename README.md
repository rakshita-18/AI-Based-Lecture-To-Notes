# 📚 AI-Based Lecture to Notes Converter

An AI-powered web application that converts lecture audio, video, or YouTube links into well-structured, easy-to-read notes using Speech-to-Text and Natural Language Processing (NLP). The application helps students save time by automatically generating summarized study notes.

## 🚀 Features

* 🎤 Convert lecture audio/video into text using AI Speech-to-Text.
* 📺 Generate notes directly from YouTube lecture links.
* 📝 AI-powered text summarization for concise and structured notes.
* 🌐 Multi-language support for transcription.
* 📄 Download generated notes as PDF.
* 📚 View previously generated notes in the History section.
* 🔐 Secure user authentication and cloud storage.

## 🛠️ Tech Stack

**Frontend**

**Backend**

**AI & NLP**
* NLP

**Database & Storage**

* Firebase Authentication
* Firebase Cloud Storage

## 📂 Project Workflow

1. User uploads a lecture audio/video or enters a YouTube URL.
2. The system extracts audio and converts speech into text.
3. AI summarizes the transcript into structured notes.
4. Users can view, save, and download the generated notes as a PDF.

## 🎯 Benefits

* Saves students' note-taking time.
* Produces clear and organized study material.
* Improves learning efficiency with AI-generated summaries.
